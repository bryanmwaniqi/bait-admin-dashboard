Souk Admin Dashboard

Version 1.0.0

Souk is a minimalistic HTML e-commerce admin dashboard template designed using bootstrap and consideration to UI and UX principles, to achieve not only an aesthetically appealing user interface but also ensure practicality to whichever developer uses it as a base for developing an e-commerce admin dashboard. With a single glimpse of the dashboard page, one is able to get a clear picture of the performance and progress of the business serving the data being laid bare and also analysed. This template can also be used as a multipurpose dashboard with a little twitching here and there, given the versatility in its design.

Circular progress bar

The circular progress bar is used in a couple of instances using SVG in this template such as;

  • Conversion rate
  • Cart abandonment rate
  • Profile completion progress
  • Skills progress
To calculate the svg circle percentage, below is the formula on how to go about it

To get the full stroke-dasharray of the circle, calculate the circumference of the circle with 2*3.14*r

To get the value to input as stroke-dashoffset, the formula is as;
stroke-dasharray - (stroke-dasharray * (%/100)) 

                                
.pie-progress-bar svg circle{
    fill: none;
    stroke-width: 0.6rem;
    stroke: #f1f1f1;
    transform: translate(10px, 10px);
    stroke-linecap: round;
    stroke-dasharray: 390;
    stroke-dashoffset: 390;
}
.pie-progress-bar svg.conversion-rate circle:nth-child(2){
    stroke: var(--primary-color);
    stroke-dashoffset: calc(390 - (390 * 73) / 100);
}

Charts

The charts in this template are courtesy of Chartjs and ApexChartsjs. The logic of all this charts is in a separate js file named charts.js located in the js folder. While there are many options to bring the charts to life, the opted option is just for demonstration purposes. Kindly make sure you familiarise yourself with the said libraries.

  • Chart.js Charts

    • The following are the charts created using Chart JS;

    • conversion rate line chart

      • This chart is used to show the rate at which visitors who make a purchase and its implemented in the analytics page.

    • cart abandonment rate line chart

      • This chart is used to show the number of shoppers who don't complete a transaction and is also implemented in the analytics page.

    • sales v expenses v profit bar chart

      • This chart shows comparison of sales, expenses and profits and is implemented in the analytics page.

    • sales bar chart

      • This chart visualises annual sales perfomance and is implemented in the index/dashboard page.

    • invoices bar chart

      • This chart visualises annual invoice issued and is implemented in the index/dashboard page.

    • returns bar chart

      • This chart visualises annual returns of orderd products and is implemented in the index/dashboard page.

    • sales line chart

      • This chart visualises annual sales perfomance but in a gradient line chart and is implemented in the sales page.

  • apexcharts.js Charts

    • The following are the charts created using ApexCharts JS;

    • Best sellers radial chart

      • The configuration of the best sellers chart is as follows; 

                                          
var options1 = {
    chart: {
        height: 280,
        type: "radialBar",
    },
    series: [67, 84, 97, 55],
    colors: ["#6630e6",  "#00cece", "#F4B400", "#ff0077"],
    plotOptions: {
        radialBar: {
        hollow: {
            margin: 35,
            size: "55%"
        },
        dataLabels: {
            name: {
                offsetY: -1,
                show: true,
                color: "#000",
                fontSize: "25px"
            },
            value: {
                color: "#888",
                fontSize: "15px",
                show: true
            },
            total: {
                color: "#000",
                fontSize: "20px",
                show: true,
                label: 'Total'
            }
        }
        }
    },
    stroke: {
        lineCap: "round",
    },
    labels: ['Fashion', 'Cosmetics', 'Electronics', 'Food & Drinks'],
    responsive: [{
        breakpoint: undefined,
        options: {},
    }]
    };
    
new ApexCharts(document.querySelector("#bestSellersChart"), options1).render();
