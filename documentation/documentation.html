Souk Marketplace

Souk Admin Dashboard

Version 1.0.0

Souk is a minimalistic HTML e-commerce admin dashboard template designed using bootstrap and consideration to UI and UX principles, to achieve not only an aesthetically appealing user interface but also ensure practicality to whichever developer uses it as a base for developing an e-commerce backend. With a single glimpse of the dashboard page, one is able to get a clear picture of the performance and progress of the business serving the data being laid bare and also analysed. This template can also be used as a multipurpose dashboard with a little twitching here and there, given the versatility in its design.
